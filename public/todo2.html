How to Use Your To-Do List App (Cloudflare KV)

Cloud Storage

  • Tasks are saved to Cloudflare KV, not your browser
  • Accessible from any device with the same user key

Add Tasks

  • Type your task in the input box at the top
  • Press Enter or click the + button to add it

Manage Tasks

  • Click the checkbox to mark tasks as complete
  • Edit or delete tasks as needed
